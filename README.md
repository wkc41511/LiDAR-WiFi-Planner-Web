# LiDAR WiFi Planner Web

Static GitHub Pages site for LiDAR WiFi Planner support and legal pages.

The visual style follows the existing `privacy-site` page, including the dark background image, glass card layout, sticky language selector, browser-language detection, and manual language switching.

- `index.html`: landing page
- `support.html`: support page
- `privacy.html`: privacy policy
- `terms.html`: terms of use
- `site.js`: localized content and browser-language routing
- `assets/privacy-bg-ink-wifi.png`: background artwork
